# **App Name**: Rota Certa

## Core Features:

- Driver Check-in: Drivers record vehicle status with details: lights, fluids, equipment, cleanliness, etc., at departure.
- Driver Check-out: Drivers record vehicle status and fuel level, and include date, odometer, and time on return.
- Vehicle Profile: Check-in and Check-out saved on each vehicle profile.
- Admin Dashboard: Administrators can view a summary of the alerts for vehicles including the current fuel levels, current issues with vehicles and maintenance requirements
- Admin Reports: Generate reports of vehicle check-in and check-out logs.

## Style Guidelines:

- Primary color: Dark Blue (#1E3A8A) to convey trust and reliability.
- Background color: Very light gray (#F5F5F5) to ensure readability and a clean interface.
- Accent color: Light blue (#7DD3FC) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif font with a modern look; suitable for headlines or body text.
- Use clear, recognizable icons for vehicle status indicators (lights, fluid levels, etc.).
- Design a dashboard layout that is easy to read. Critical vehicle information should be at the top.
- Subtle animations on form submissions to acknowledge data capture.
